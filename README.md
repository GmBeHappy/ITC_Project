# Simple_cashless_system_simulation

การจำลองระบบการใช้จ่ายไร้เงินสดด้วย arduino

โปรเจคนี้เป็นงานของวิชา Introduction to Computer Engineering 

สามารถดูคลิปการทำงานได้ที่ [here](https://youtu.be/lD12LQloVX0)

## Getting started
### อุปกรณ์ที่ใช้

1. Arduino UNO
2. RFID RC522
3. LCD Display
4. OLED Display
5. I2C EEPROM 
6. Passive Buzzer
7. LDR
8. Button
